# Python Calculator

파이썬으로 만든 간단한 계산기 프로젝트입니다.

## 브랜치 정보

- 수정 브랜치: **dev**
- 설명: 개발·테스트용 문서를 dev 브랜치에서 업데이트했습니다.

## 기능

- **더하기** (`add`): 두 수의 합을 반환합니다.
- **빼기** (`subtract`): 두 수의 차를 반환합니다.

## 사용 예

```python
from calculator import add, subtract

add(10, 3)       # 13
subtract(10, 3)  # 7
```

## 실행

```bash
python calculator.py
```
